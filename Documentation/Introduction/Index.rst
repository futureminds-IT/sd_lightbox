Introduction
================

This chapter gives you a basic introduction about the TYPO3 CMS extension "sd_lightbox".


What does it do?
----------------

This extension integrates a responsive and touch-friendly lightbox into TYPO3.

- Based on extbase & fluid
- Responsive and touch-friendly
- Needs jQuery and hammer.js
- Uses CSS transforms and transitions
- Interacts with keyboard
- Fully configurable via constants
- Tested in modern Browsers (Edge, Firefox, Chrome, Safari, Opera, IOS, Android and more)
